# Listado de Herramientas y Librerías

Para el desarrollo de este componente Smart Home, se han utilizado las siguientes herramientas:

### Herramientas de Diseño
Scene Builder: Utilizado para realizar el prototipado visual inicial de la disposición de los nodos del componente.

### Herramientas de Desarrollo
Eclipse IDE: Entorno de desarrollo principal para la escritura del código Java y la gestión del proyecto.
GitHub: Sistema de control de versiones utilizado para el seguimiento del código y el alojamiento del repositorio.
Gradle: Herramienta de automatización utilizada para gestionar la compilación, dependencias y el empaquetado del componente.

### Herramientas de Prueba
JUnit 5: Framework para la realización de pruebas unitarias sobre la lógica de las funciones del componente.
TestFX: Librería especializada para automatizar las pruebas de interacción de usuario en interfaces JavaFX.

### Librerías
JavaFX SDK: El framework base para el desarrollo del componente personalizado, sus propiedades y animaciones.
