# Listado de Herramientas y Librerías

Para el desarrollo de este componente Smart Home, se han utilizado las siguientes herramientas:

### Herramientas de Diseño
* [cite_start]**Scene Builder**: Utilizado para realizar el prototipado visual inicial de la disposición de los nodos del componente[cite: 76].

### Herramientas de Desarrollo
* [cite_start]**Eclipse IDE**: Entorno de desarrollo principal para la escritura del código Java y la gestión del proyecto[cite: 76].
* [cite_start]**Git / GitHub**: Sistema de control de versiones utilizado para el seguimiento del código y el alojamiento del repositorio[cite: 76].
* [cite_start]**Gradle**: Herramienta de automatización utilizada para gestionar la compilación, dependencias y el empaquetado del componente[cite: 76].

### Herramientas de Prueba
* [cite_start]**JUnit 5**: Framework para la realización de pruebas unitarias sobre la lógica de las funciones del componente[cite: 76, 79].
* [cite_start]**TestFX**: Librería especializada para automatizar las pruebas de interacción de usuario en interfaces JavaFX[cite: 28, 76].

### Librerías
* [cite_start]**JavaFX SDK**: El framework base para el desarrollo del componente personalizado, sus propiedades y animaciones[cite: 76].
